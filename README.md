# inuMixer

![inumixer](https://i.imgur.com/IpO5ivg.gif "image")

ダウンロード:  **[inuMixer v1.0.1](https://github.com/masahiroitojapan/inuMixer/releases/download/v1.0.1/inuMixer.v1.0.1.zip)**


inuMixer は、Windowsでのオーディオ体験を快適にする、DAW風の縦型ボリュームミキサーです。<br>
「ゲーム中に裏で鳴っているDiscordやブラウザの音量を、サッと調整したい」という個人的なニーズから生まれました。<p>

<details>
<summary>心の声</summary>
楽しみだった新作ゲームを購入した時、初起動の際にしばしば起こる問題としてシステムコンフィグ画面を設定する前に稲妻のようなバカでっっっっけぇ音量でワクワクするようなオープニングムービーが流れるようなゲームってあるよね。
このアプリケーションはアプリケーションシステムの音量を調整するので、ゲームがフェーダーに追加されたらマウススクロールでちょっと音量を下げておけば突然の大音量も怖くないってワケ。
</details>


### ✨ 主な機能

- フェーダーUI: 直感的で視認性の高いスライダー操作。
- アプリごとの音量管理: 起動中のアプリケーションを自動検知し、個別に音量調整・ミュートが可能。
- マスターフェーダー: システム全体の音量を一括コントロール。
- ドラッグ＆ドロップ並び替え: フェーダーの位置を自由に入れ替え可能。配置は次回起動時にも記憶されます。
- 表示フィルター: 不要なアプリ（システム音など）を非表示設定できます。
- オーバーレイ表示:
  - 常に手前に表示: ゲームや作業画面の上に常駐可能。
  - 透明度調整: 邪魔にならないよう、ウィンドウの透過率を変更可能。


### 🎮 操作方法
- 音量調整: フェーダーの上で マウスホイールをスクロール します。（スライダーのドラッグ操作も可能）
- ミュート切替: フェーダーの上で マウスホイールをクリック（中クリック） します。（下のMuteボタンも可能）
- 配置変更: フェーダーを 左ドラッグ して好きな場所に移動します。
- 設定: 右上の ⚙️ アイコンから、表示・非表示にするアプリを選択できます。
